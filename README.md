# autocow

Display a cow every 2 second in your terminal.
Based on [cows](http://npm.im/cows) and [vaca](http://npm.im/vaca).

![demo](https://raw.githubusercontent.com/mcollina/autocow/master/autocow.gif)

## Install

```js
npm i autocow -g
```

## Usage

in a terminal: `cows`

in your node.js script:

```js
require('autocow')()
```
